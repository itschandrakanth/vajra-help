# How to Create Lead Funnel?
&nbsp;
![enter image description here](https://i.ibb.co/DDwLYJd/create-funnel-1.png)

1) First access the "Leads" page and make sure each lead has been assigned a status.

![enter image description here](https://i.ibb.co/kK23nbt/create-funnel-2.png)

2) Next, switch to "Lead Funnel" page and scroll down to "Funnel" section.

![enter image description here](https://i.ibb.co/zx3Swz5/create-funnel-3.png)

3) In the "Funnel" section tap on "Edit Funnel" button.

![enter image description here](https://i.ibb.co/B2HbDNS/create-funnel-4.png)

4) Select all options and click "Submit" button.

![enter image description here](https://i.ibb.co/chskfW6/create-funnel-5.png)

5) Now your account's leads funnel will appear in the "Funnel" section.
